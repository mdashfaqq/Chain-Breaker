# Chain-Breaker

## Automated SOC Platform for Real-Time Threat Detection and Response

Chain-Breaker is a Security Operations Center (SOC) platform that detects, correlates, and responds to cyber attacks automatically.

### Technology Stack

* Suricata IDS
* Wazuh SIEM
* Elasticsearch & Kibana
* Keycloak IAM
* Shuffle SOAR
* TheHive Incident Response

### Attack Scenarios

* Nmap Reconnaissance
* SQL Injection
* Brute Force Attacks

### Security Pipeline

Kali Linux
↓
DVWA
↓
Suricata
↓
Wazuh
↓
SOAR
↓

Automated Response
